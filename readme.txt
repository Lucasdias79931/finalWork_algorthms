Trabalho final de algoritmos

Esse é o Trabalho final da matéria de algoritmos;
objetivo de usar todos as ferramentes aprendidas como funcoes, manipulaçõeos de strings e matrizes, laços de repetições 
e controle de fluxo de código com condicionais para criar um mini jogo.




                                            ALGORITMO:
    0 - Inicio
    1 - Criar uma matriz 6x6
    2 - numa função, inicialize a matriz com números randomicos
    3 - crie as variaveis de controle de fluxo
        o jogo irá rodar em um loop while(true) até o fim
    4 - crie a funções:
        4.1 display e menu para atualizar as informações para o usuário
        4.2 criar funcao operação
    5 - inicia o loop principal
        5.1 pede as coordenadas ao usuário
            5.1.1 faz o tratamento para verificar se apenas números foram digitados 
        5.2 solicita a operacao 
            5.2.1 5.2.1 faz o tratamento para ver se digitou apenas letra
        5.3 solicita ao usuário qual ele acha que é o resultado da operação
            5.3.1 faz o tratamento para ver se digitou apenas numeros e decimal
